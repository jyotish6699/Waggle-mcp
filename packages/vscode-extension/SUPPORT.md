# Support

For bugs, installation problems, and feature requests:

- GitHub issues: <https://github.com/Abhigyan-Shekhar/Waggle-mcp/issues>
- Install docs: <https://github.com/Abhigyan-Shekhar/Waggle-mcp/tree/main/docs/install>

When reporting a problem, include:

- your operating system
- VS Code version
- extension version
- whether `waggle-mcp doctor` succeeds
- relevant output from the `Waggle` output channel
